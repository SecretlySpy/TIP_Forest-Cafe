# Module / File: index.html

## Function: Main User Interface
- **Purpose**: Defines the structural layout and content of the Forest Cafe landing page.
- **Inputs**:
  - `None` (`N/A`): Static HTML file.
- **Outputs**: Rendered DOM tree representing the web page.
- **Dependencies**: `style.css`, Google Fonts (Inter).
- **Behavior**: 1. Loads external fonts and stylesheet. 2. Renders a centered main application card. 3. Displays the cafe header, a list of daily specials with prices, and an order button.
- **Side Effects**: None

# Module / File: style.css

## Function: Application Styling
- **Purpose**: Provides the visual presentation, theming, and responsive layout for the application.
- **Inputs**:
  - `None` (`N/A`): Static CSS file.
- **Outputs**: CSS Object Model (CSSOM) rules.
- **Dependencies**: None.
- **Behavior**: 1. Defines CSS custom properties (variables) for the forest color palette and shadows. 2. Resets default browser margins/padding. 3. Styles the main app container using flexbox for centered alignment. 4. Applies hover states, transitions, and media queries for mobile, tablet, and desktop viewports.
- **Side Effects**: None
